## Datasets related to UK Geographies

The following datasets are available both in *CSV* format, in the `data-raw` directory, and in *RDA* format in the `data` directory:

 - **postcodes**
 
 - **output_areas**
 
 - **locations** 
 
 - **location_types**
 
 - **lookups**
 
 - **hierarchies** 
 
 - **workplace_zones**
 
 - **neighbours**
